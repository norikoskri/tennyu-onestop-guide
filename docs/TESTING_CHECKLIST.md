# 動作確認チェックリスト

## 確認方法
開発サーバー起動: `npm run dev`
アクセス: http://localhost:3001 (ポート3000が使用中の場合は3001)

---

## 1. オンボーディング（トップページ /）

### 1.1 初回アクセス
- [ ] チャット画面が表示される
- [ ] 「はじめまして！」のウェルカムメッセージが表示される
- [ ] 入力欄とクイックアクションが表示される

### 1.2 情報入力フロー
- [ ] 郵便番号入力で住所が補完される（例: 1540001 → 世田谷区池尻）
- [ ] 転入予定日を自然言語で入力できる（例: 来月15日）
- [ ] 家族構成を選択できる
- [ ] 確認画面が表示される
- [ ] 「登録する」でダッシュボードへ遷移する

---

## 2. ダッシュボード（/dashboard）

### 2.1 表示
- [ ] 進捗サマリーが表示される（完了数/全体数）
- [ ] プログレスバーが表示される
- [ ] 次のおすすめアクションが表示される
- [ ] 「あなたへのおすすめ」セクションが表示される
- [ ] タスク一覧が表示される

### 2.2 機能
- [ ] タスクを完了にできる（チェックマーク）
- [ ] タスクをタップで詳細へ遷移する
- [ ] ボトムナビゲーションで他のページへ遷移できる

### 3.2 機能
